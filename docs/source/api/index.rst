Python API Reference
====================

The Python API reference is organized into several sections:

- ``netrc``: An introduction for reading and writing ``.netrc`` file, which are used to store login information for various services.
- ``downloader``: A module contains various functions for downloading files from the web.
- ``parse_urls``: A module for parsing URLs from various sources.
